## Installation Instructions
1. Install Node v20.14.0
2. Install Angular 17:    
`npm install --save-dev @angular/cli@17`
3. Run the following in your terminal:   
`npm install`
4. Run the frontend dev build (serves at http://localhost:4200):   
`npx ng serve --open`
